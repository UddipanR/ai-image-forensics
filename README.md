# 🔬 AI Image Forensics

> A forensic analysis system that detects whether an image was created by a human or generated by Artificial Intelligence — using six deep learning models across two training domains.

---

## 📌 What This Project Does

Anyone can now create realistic AI-generated images in seconds using tools like DALL-E, MidJourney, or Stable Diffusion. This makes it very hard to tell what is real and what is fake.

This project builds a forensic tool that:
- Analyzes any uploaded image using **6 trained models**
- Works across **2 domains** — synthetic benchmark images and real-world photographs
- Shows a **heatmap** of which parts of the image influenced the decision most
- Provides a **detailed forensic explanation** of why the image appears real or AI-generated
- Ranks models by their reliability for each specific image
- Gives **no forced single verdict** — the user interprets the results

1. **Upload any image** — drag and drop, paste from clipboard, or use webcam

   - The **ranking table** shows which model is most confident and why
   - The **heatmap** shows which parts of the image the model focused on
   - The **forensic explanation** describes what signals were detected
   - The **raw scores** show the exact probability from each model

## 🧠 How It Works

### Two Preprocessing Paths

Because the models were trained on different types of images, two preprocessing methods are used:

**Synthetic Domain:**
The CIFAKE dataset originally contains 32×32 pixel images. To match this, uploaded images are:
1. Shrunk down to 32×32 pixels
2. Scaled back up to 224×224 pixels

This simulates the same pixel pattern the synthetic models learned from.

**Photographic Domain:**
The GRAVEX-200K dataset contains high quality face photographs. Images are:
1. Directly resized to 224×224 pixels

No downscaling — preserving the full detail that the photographic models expect.

### Ranking System

Each model gets a **rank score** calculated as:

```
Rank Score = Model Accuracy × Confidence Strength
```

Where confidence strength measures how far the model's prediction is from 50% (uncertain). A model that says "99% AI" gets a much higher rank score than one that says "53% AI". This means **high confidence from accurate models carries the most weight**.

### Grad-CAM Heatmap

The activation heatmap is generated using gradient saliency from the Custom CNN model. It highlights:
- 🔴 **Red and yellow zones** — areas the model found most suspicious or most authentic
- 🔵 **Blue zones** — areas that did not strongly influence the decision

## ⚠️ Important Limitations

This tool is designed for research and education. Please keep these limitations in mind:

- **Heavily edited images** — strong filters, HDR effects, or AI-enhanced retouching can confuse the models
- **Screenshots** — rephotographing a screen removes the original noise patterns the model relies on
- **Compressed images** — low quality JPEG images lose detail and may be misclassified
- **AI-enhanced real photos** — images that are partly real and partly AI-processed sit in a grey area
- **Artistic images** — hand-drawn artwork, paintings, and cartoons were not part of the training data
- **Not for legal use** — results should always be combined with human judgement
